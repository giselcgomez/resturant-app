## How to Run:
- 1st time you are running it
- Download all files to a directory

### FRONT END
In terminal type:
* > cd restaurant-app
* > npm install
* > npm run dev

- You should be able to open the app in localhost:3000

### BACKEND
### In terminal type:
* > cd backend
* > nvm use 12.0.0
* > npm install
* > npm run build
* > npm run develop

### After the first run use the following for the Backend
* > nvm use 12.0.0
* > npm install
* > npm run start

- backend: http://localhost:1337
